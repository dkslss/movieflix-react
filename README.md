# Movieflix

넷플릭스 클론 프로젝트 입니다.  
React + Tailwind CSS 로 스타일링 & GitHub Pages 배포 완료!

##  기술스택
- React (Create React App)
- Tailwind CSS
- React Router
- gh-pages (GitHub Pages 배포)
- Open API 사용 : YTS API (https://yts.mx/api)
